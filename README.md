#### Solution to problem [Bot Trust](https://code.google.com/codejam/contest/975485/dashboard) written in Haskell for a university assignment.

**Instructions to run script using GHC 8.2.2 & BASH 4.4.**

- Compile script.

	ghc bottrust.hs

- Run script.

	./bottrust

- Enter path to one of the inputs files (these can be downloaded from the link above).

- Result will be saved in file res.txt, that is created in the dir where the executable resides.
